교육 시간
- 월 : 13~18시
- 화~금 : 10~18시

지각/결석
- 5분 초과 : 지각
- 30분 초과 : 결석
- 등교하지 않은 경우, 결석
- 결석 횟수 5회 : 제적
- 지각 3회 = 결석 1회

대상자
- 경고 대상자 : 결석 2회 이상
- 면담 대상자 : 결석 3회 이상
- 제적 대상자 : 결석 5회 초과

운영시간
: 08~23시
- 주말, 공휴일은 출석 X

1. 출석 확인
- 닉네임, 등교 시간 입력
- 출석 후 출석 기록 확인

2. 출석 수정
- 닉네임, 수정 날짜, 등교 시간 입력
- 변경 전과 변경 후의 출석 기록 확인

3. 크루 별 출석 기록 확인
- 닉네임 입력
- 전 날까지의 출석 기록 확인
  - 출석, 지각, 결석
  - 대상자 여부

4. 제적 위험자 확인
- 전날까지의 출석 기록을 바탕으로, 제적 위험자 파악
- 제적 대상자, 면담 대상자, 경고 대상자 순으로 출력
  - 지각을 결석으로 간주하여 내림차순 (지각 + 결석의 수)
  - 출석 상태가 같으면, 닉네임으로 오름차순
