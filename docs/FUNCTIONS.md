1. 메인 화면
- [X] 오늘 날짜 출력하기
- [X] 기능 선택 입력 받기
  - [X] [ERROR] 잘못된 값 입력했을 경우 (1, 2, 3, 4, Q가 아닌 경우)

2. [1] 출석 확인
- [X] 평일인지 확인하기
  - [X] 평일이 아니라면, [ERROR] 등교일이 아니라고 출력하기
- [X] 출석 닉네임 입력 받기
  - [X] 등록된 이름인지 확인하기
    - [X] 등록된 이름이 아니라면, [ERROR] 등록되지 않은 이름이라고 출력하기
  - [x] 이미 출석된 이름인지 확인하기
    - [X] 이미 출석된 경우라면, [ERROR] 출석된 이름이라고 출력하기
- [ ] 등교 시간 입력 받기
  - [ ] 형식 확인하기
    - [ ] 잘못된 형식이라면, [ERROR] 잘못된 형식이라고 출력하기
  - [ ] 운영 시간인지 확인하기
    - [ ] 운영 시간이 아니라면, [ERRPR] 운영 시간에만 출석 가능하다고 출력하기
- [ ] 출석 완료 결과 출력하기

3. [2] 출석 수정
- [ ] 닉네임 입력 받기
  - [ ] 등록된 이름인지 확인하기
    - [ ] 등록된 이름이 아니라면, [ERROR] 등록되지 않은 이름이라고 출력하기
- [ ] 수정 날짜 입력 받기
  - [ ] 형식 확인하기
    - [ ] 잘못된 형식이라면, [ERROR] 잘못된 형식이라고 출력하기
  - [ ] 과거인지 확인하기
    - [ ] 잘못된 날짜라면, [ERROR] 수정할 수 없다고 출력하기
- [ ] 등교 시간 입력 받기
  - [ ] 형식 확인하기
    - [ ] 잘못된 형식이라면, [ERROR] 잘못된 형식이라고 출력하기
  - [ ] 운영 시간인지 확인하기
    - [ ] 운영 시간이 아니라면, [ERRPR] 운영 시간에만 출석 가능하다고 출력하기
- [ ] 수정 전과 후 결과 출력하기

4. [3] 크루 별 출석 기록 확인
- [ ] 닉네임 입력 받기
  - [ ] 등록된 이름인지 확인하기
    - [ ] 등록된 이름이 아니라면, [ERROR] 등록되지 않은 이름이라고 출력하기
- [ ] 출석 기록 출력하기
  - [ ] 출석 기록 정보 출력하기
  - [ ] 출석, 지각, 결석 횟수 출력하기
  - [ ] 경고, 면담, 제적 대상자 여부 출력하기

5. [4] 제적 위험자 확인
- [ ] 조회 결과 출력하기
  - [ ] 지각을 결석으로 간주하여, 결석과 지각 횟수로 내림차순 정렬하여 출력하기
  - [ ] 결석 횟수가 같으면, 닉네임 기준으로 오름차순 정렬하여 출력하기
